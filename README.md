# Frappe Notification Bridge

Bridge Frappe notifications to MS Teams chat notifications for 1:1 user interaction.

## Features

- Parse all system notifications.
- Send as 1:1 MS Teams chatbot messages.
- User mapping via login email / Office 365 SSO.
- Deep links back to ERPNext/Frappe.

## Setup

1.  Install the app.
2.  Configure MS Teams Bot credentials.
3.  Map your users.
